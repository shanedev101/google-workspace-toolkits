# Privacy Policy

Last updated: May 26, 2026

Workspace Toolkit for Google is a browser extension built by **shanedev101**.

## Data collection

This extension collects no data. It does not send your documents, spreadsheet content, email drafts, or any other information to any server. There are no analytics, no tracking scripts, and no third-party services involved.

## Local storage

The extension uses Chrome's `storage` API to save your preferences (theme, toggle states, integration settings) locally on your device. This data stays on your machine and is never transmitted anywhere.

## Network access

The extension makes no outbound network requests. All rendering — Markdown parsing, syntax highlighting, math formulas, diagrams — runs entirely inside your browser using bundled libraries.

## Open source

The full source code is available under the MIT License. You can audit exactly what the extension does.
