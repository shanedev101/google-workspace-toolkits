Google Docs

Select markdown text in Docs to show a floating preview button and render in the sidebar.

Formatting Guide

Markdown Support

  • Headings: # H1, ## H2
  • Styles: **bold**, *italic*
  • Lists: - Item, 1. Item
  • Tables: Standard GF Markdown tables

Math (KaTeX)

  • Inline Math: $E = mc^2$
  • Block Math: $$ \text{Total} = \sum_{i=1}^n x_i $$

Mermaid Diagrams

  • Use ```mermaid block inside selection
